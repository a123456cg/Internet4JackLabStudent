### 如何取得

1. 下載專案：
    ```bash
    git clone https://github.com/RemakeAONTeam/AON.git
    ```

2. 取得大型檔案：

    ```bash
    git lfs fetch
    ```

    ```bash
    git lfs pull
    ```
