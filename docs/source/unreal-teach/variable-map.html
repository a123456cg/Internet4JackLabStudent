I.II. Variable Map

Variable Map 設定和呼叫

設定Variable Map的子Map

Damage Map

  1. 工具列點擊Class Defaults
  2. 找到MOBA下的Variable Map
  3. 按下Variable Map右邊的+ , 新增一個名稱可自定義的Map
  4. 此Map填上Damage
  5. 展開Damage Map底下的Values找到右邊的+ ,新增技能等級數個Value
  6. 填上各等級的傷害

Other Map

  • 其他特殊效果只要在(4.)填上不同名稱;以及在(6.)填上不同效果值即成為其他效果Map

  • 例如Critical Map(爆擊率):

在主圖 (EventGraph) 視窗中呼叫 Variable Map 的子 Map

以 Damage Map 為例
  1. 右鍵新增Variable Map節點, 右拉出Find節點, Find節點輸入Damage即取得Damage Map
  2. 右鍵Get Current Level
  3. Current Level節點右拉輸入-(減號)得到interger - interger運算元節點, 下面輸入1表示減1
  4. Find節點右拉輸入Break新增Break LevelVariable把Damage Map的參數分開成float array, 此array右拉輸入Get新增Get節點, 此Get節點依照Index = Current Level-1取出Array裡的值
